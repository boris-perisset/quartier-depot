Menu File → Download → Comma-separated values

Choose whatever name you wish (not important), and save it somewhere.

Click the button below to load the CSV file into this page.

Wait several seconds for all labels to be automatically generated :-)

Press Hide instructions to hide extra text before printing (reload to reset).

Press File &rarr; Print to PDF or generate a PDF for all labels.

Select to print "2 pages per sheet".

In More settings activate option Background graphics, so colors are printed right.

Verify that the labels are ~10cm in length, or 1/3 of the sheet (A4 = 21x30cm)

Print, cut and trim labels onto the racks.
